# Tableau Project - Wish E-commerce Summer Sales Data Dashboard
## Dataset
This dataset contains information about summer product listings, ratings, and sales performance data on the [Wish e-commerce platform](https://www.wish.com/).
## Objective
Use Tableau to breakdown the dataset to gain insights.
